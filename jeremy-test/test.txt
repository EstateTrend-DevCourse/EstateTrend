PR방법
1.원본 주소로 가서 FORK딴다
2.원하는 폴더에 repository 클론하기
3. remote url 추가
git remote add <remote url 이름> <remote url>
git remote -v -> 상태확인
4. branch 생성
git branch <branch 이름>
git branch -> branch list 볼수있음
5. branch 이동
git checkout <branch 이름>
6. 코드 수정하기
git add .  >> all
git add <파일 이름> >> 특정 파일만
